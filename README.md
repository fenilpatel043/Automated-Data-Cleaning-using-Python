Automated Data Cleaning using Python
Overview

This project automates the data cleaning process using Python, converting raw, messy datasets into analysis-ready data with minimal manual effort.

 What This Automation Does
Normalizes missing values (na, null, blanks → NaN)
Fills missing numerical data using median
Fills missing categorical data using forward & backward fill
Fixes incorrect data types
Removes duplicates
Outputs clean, structured data

Tech Stack
Python
Pandas
NumPy
Jupyter Notebook

Why This Project
Shows real-world data preprocessing, automation mindset, and readiness for SQL & Power BI analysis — not just basic Pandas usage.
