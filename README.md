Automated Data Cleaning using Python

Overview
This project automates the data cleaning process using Python.  
It transforms raw, messy datasets into clean, analysis-ready data with minimal manual effort.



Features
- Normalizes missing values (`na`, `null`, blanks â†’ NaN)
- Handles missing numerical values using median
- Handles missing categorical values using forward & backward fill
- Corrects data types
- Removes duplicate records
- Prepares data for further analysis and visualization



Tech Stack
- Python
- Pandas
- NumPy
- Jupyter Notebook


Project Files
Automating data cleaning.ipynb`



How to Run
1. Clone the repository  
2. Open the notebook in Jupyter  
3. Update the dataset path if required  
4. Run all cells to generate cleaned data  



Use Case
Useful for Data Analysts and Business Analysts to automate repetitive data preprocessing tasks before using SQL, Power BI, or other analytics tools.
